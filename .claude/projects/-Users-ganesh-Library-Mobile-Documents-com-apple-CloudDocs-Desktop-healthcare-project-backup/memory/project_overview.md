Healthcare Management System with a Django REST Framework backend and React 19 frontend.

**Backend apps:** users, patients, doctors, appointments, analytics
**Auth:** JWT via simplejwt, custom User model (`users.User`)
**API docs:** Swagger/ReDoc at `/swagger/` and `/redoc/`
**Database:** SQLite (dev)
**Frontend:** React 19 + Bootstrap 5, axios for API calls, react-router-dom v7

**Why:** Early-stage project (2 commits on main), actively being built out.

**How to apply:** This is a full-stack app — backend runs on Django dev server, frontend via `react-scripts start`. API is under `/api/` prefix.
